大学院生募集中

メダカ・ゼブラフィッシュを用いた体軸形成・神経発生・器官発生の研究に興味のある大学院生を募集しています。

当研究室には、名古屋大学大学院理学研究科生命理学専攻の動物器官機能学グループの修士（博士前期）過程および博士課程の学生として研究に携わることができます。
大学院入試に関しては、生命理学専攻のホームページをご覧ください。 研究室にご興味のある方は、ご連絡ください。
